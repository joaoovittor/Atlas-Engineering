## Atlas Engineering

A front-end study project focused on building a professional multi-page website layout using CSS Grid and Flexbox.

## ATL Engineering
ATL Engineering is a fictional construction and engineering company website created to practice complex layouts, multi-page navigation, and modern UI structure.

The project simulates a professional construction company website, showcasing residential and apartment developments, engineering services, pricing plans, and contact sections designed to represent a real-world corporate interface.

## Pages
- Home
- Projects
- Services
- Contact
  
## Technologies
- HTML  
- CSS (Grid & Flexbox)

## About
This project was built for learning and practice purposes, focusing on multi-page structure, layout composition, responsive sections, and combining CSS Grid with Flexbox to organize complex website layouts.
